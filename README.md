# Big Data Frijoles
## Descripción: 
Proyecto que abarca el análisis de Big Data a un conjunto de datos para granos de Frijoles.

## Enfoque de desarrollo:
El proyecto utilizó la metodología CRISP-DM. 

## Tecnologías:
Se utilizó el lenguaje de programación Python.
